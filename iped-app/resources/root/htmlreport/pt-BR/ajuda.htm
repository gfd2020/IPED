Armazenamento e visualização dos arquivos

  • Os arquivos selecionados durante os exames foram renomeados e exportados para o diretório "Exportados" desta mídia. Para obter os nomes e demais informações originais dos arquivos acesse "Marcadores".

  • Recomenda-se configurar o programa navegador para modo de trabalho offline, de forma que o conteúdo de arquivos hipertexto, tais como temporários de navegação na Internet, seja visualizado sem atualização de dados em servidores externos.

  • Para visualizar o conteúdo desta mídia ótica num computador com sistema Linux/Unix, certifique-se de que as configurações de locales do sistema estão configuradas para o conjunto de caracteres ISO-8859-1.

  • Nem todos os arquivos exportados nesta mídia podem ser abertos diretamente pelo programa navegador utilizado. Neste caso, pode ser necessária a instalação do aplicativo apropriado (entre em contato com o suporte técnico de informática do seu setor para informações sobre visualizadores disponíveis).



O Algoritmo SHA-1

  • O SHA-1 (Secure Hash Algorithm de 160 bits) é um algoritmo que, a partir de uma mensagem de entrada de qualquer tamanho, gera uma saída de tamanho fixo de 160 bits (conhecido como código de integridade, resumo ou hash), calculada a partir do conteúdo dessa mensagem. A segurança do procedimento consiste no fato de não ser conhecido método computacionalmente viável para produzir o mesmo código de integridade a partir de duas mensagens distintas ou, a partir do código de integridade, obter a mensagem de entrada.

  • Cada arquivo contido nesta mídia ótica é tratado como se fosse uma mensagem que passa individualmente pelo processamento do algoritmo. Ao final, obtém-se a relação dos nomes dos arquivos precedidos por seus respectivos códigos de integridade em formato hexadecimal.

  • O acréscimo, alteração ou remoção de um único caractere em um arquivo é condição suficiente para que o código de integridade gerado seja diferente, tornando detectável a alteração do conteúdo desta mídia ótica.
